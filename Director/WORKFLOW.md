# 🔄 Бизнес-логика и сценарии работы бота

## Общая схема взаимодействия

```
Пользователь -> /start -> Главное меню -> Выбор режима:
  ├─ 📝 Оставить заявку
  ├─ 💬 Консультация с AI
  └─ ❓ Помощь
```

## Сценарий 1: Оставить заявку

### Состояния FSM (Finite State Machine)

```
APPLICATION_FLOW:
  1. project_type      - Тип проекта
  2. budget_range      - Бюджет
  3. timeline          - Сроки
  4. contact_info      - Контактная информация
  5. task_description  - Описание задачи (свободная форма)
  6. confirmation      - Подтверждение
```

### Детальный поток

```
User: /start
Bot: [Inline кнопки: 📝 Заявка | 💬 Консультация | ❓ Помощь]

User: Нажимает "📝 Заявка"
Bot: Отлично! Помогу оформить заявку. Какой тип проекта вас интересует?
     [Inline кнопки: Бот | CRM | Сайт | Парсинг | Другое]

User: Выбирает "Бот"
Bot: Какой функционал нужен у бота?
     [Inline кнопки: Рассылка | Сбор заявок | Интеграция | Другое]

User: Выбирает "Сбор заявок"
Bot: Понял! Какой бюджет планируете?
     [Inline кнопки: до 50к | 50-100к | 100-200к | 200к+]

User: Выбирает "50-100к"
Bot: Когда планируете начать?
     [Inline кнопки: На этой неделе | В этом месяце | В течение квартала | Планирую]

User: Выбирает "В этом месяце"
Bot: Как с вами связаться? (телефон или email)

User: Отправляет "+79991234567"
Bot: Расскажите подробнее о задаче. Что именно должен делать бот?
     (свободная форма)

User: Описывает задачу
Bot: ✅ Заявка оформлена! 
     
     📋 Резюме:
     Тип: Бот для сбора заявок
     Бюджет: 50-100к руб
     Сроки: В этом месяце
     Контакт: +79991234567
     
     Наш менеджер свяжется с вами в ближайшее время!
     
     [Inline кнопки: 💬 Консультация с AI | 🏠 Главное меню]
```

## Сценарий 2: Консультация с AI

### Состояния FSM

```
CONSULTATION_FLOW:
  1. active_consultation  - Активный диалог с AI
  2. waiting_ai_response  - Ожидание ответа от AI
```

### Детальный поток

```
User: /start
Bot: [Inline кнопки: 📝 Заявка | 💬 Консультация | ❓ Помощь]

User: Нажимает "💬 Консультация"
Bot: 💬 Запускаю режим консультации с AI-специалистом.
     
     Опишите ваш вопрос или задачу, и я постараюсь помочь 
     с рекомендациями по автоматизации!

User: Хочу автоматизировать отчеты
Bot: [AI обрабатывает и отвечает]
     Отличная цель! Автоматизация отчетов - одна из самых 
     востребованных задач.
     
     Подскажите, из каких источников обычно берутся данные 
     для отчетов? (Excel, Google Sheets, CRM, сайт, другое)

User: Из Google Sheets и CRM
     
Bot: [AI продолжает диалог...]
     ...
     
     [В любой момент:]
User: Хочу оставить заявку
Bot: [Переключает на сценарий заявки]
```

## Сценарий 3: Отмена действия

```
[В любом состоянии]
User: /cancel
Bot: ❌ Действие отменено.
     
     [Inline кнопки: 📝 Заявка | 💬 Консультация | 🏠 Главное меню]
```

## Сценарий 4: Помощь

```
User: /help или нажал "❓ Помощь"
Bot: 📖 Справочная информация

     Этот бот помогает:
     📝 Оставить заявку на разработку
     💬 Получить консультацию по автоматизации
     
     Команды:
     /start - Главное меню
     /help - Справка
     /cancel - Отменить текущий диалог
     
     [Inline кнопки: 📝 Заявка | 💬 Консультация | 🏠 Главное меню]
```

## Обработка ошибок

### Пользователь отправил некорректные данные
```
User: [В поле бюджета отправил "не знаю"]
Bot: ⚠️ Пожалуйста, выберите бюджет из предложенных вариантов 
     или напишите "не определился"

     [Inline кнопки: до 50к | 50-100к | 100-200к | 200к+ | Не определился]
```

### AI сервис недоступен
```
User: [В режиме консультации]
Bot: ⚠️ В данный момент AI-консультант временно недоступен.
     
     Пожалуйста, попробуйте позже или оставьте заявку - 
     наш специалист свяжется с вами!
     
     [Inline кнопки: 📝 Оставить заявку | 🏠 Главное меню]
```

### Таймаут сессии
```
[Пользователь не отвечал 30 минут]
Bot: ⏰ Сессия неактивна. Если хотите продолжить, нажмите:
     
     [Inline кнопки: 📝 Начать заявку заново | 💬 Новая консультация | 🏠 Главное меню]
```

## Переходы между состояниями

```
Главное меню
  ├─> APPLICATION_FLOW (через "📝 Заявка")
  │     ├─> CONFIRMATION -> Главное меню
  │     └─> /cancel -> Главное меню
  │
  ├─> CONSULTATION_FLOW (через "💬 Консультация")
  │     ├─> User просит заявку -> APPLICATION_FLOW
  │     └─> /cancel -> Главное меню
  │
  └─> HELP (через "❓ Помощь")
        └─> Главное меню
```
