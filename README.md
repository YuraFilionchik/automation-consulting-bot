# 🤖 AI Bot для консультаций и приема заявок

## Описание проекта
Telegram-бот с AI-интерфейсом для:
1. **Приема заявок от клиентов** - через уточняющие вопросы собирает информацию о проекте
2. **Консультаций с AI агентом** - имитирует специалиста по автоматизации

## Технологический стек
- **Python 3.10+** - основной язык
- **aiogram 3.x** - асинхронная библиотека для Telegram Bot API
- **OpenAI API / LiteLLM** - для AI функционаности (поддержка разных моделей)
- **SQLite** - легковесная БД (для маломощных серверов)
- **Redis** - кэширование и состояние диалогов (опционально)
- **systemd** - управление сервисом на Linux

## Основные сценарии

### 1. Режим приема заявок
```
/start -> Выбор "Оставить заявку" -> Серия уточняющих вопросов:
  - Какой тип проекта? (сайт/бот/CRM/другое)
  - Бюджетные рамки?
  - Сроки?
  - Контакты?
  - Описание задачи (свободная форма)
-> Заявка сохранена -> Подтверждение клиенту
```

### 2. Режим консультации
```
/start -> Выбор "Консультация с AI" -> AI диалог:
  - AI задает уточняющие вопросы
  - Дает рекомендации по автоматизации
  - Может предложить оставить заявку
```

## Требования к серверу
- **Минимальные**: 1 CPU, 512MB RAM
- **Рекомендуемые**: 1 CPU, 1GB RAM
- **OS**: Ubuntu 22.04 LTS
- **Хранилище**: 10GB+
